## 🎯 快速验证脚本

### 创建一个test_my_algorithm.py文件：
```python
from n_queens_solver import NQueensSolver
from advanced_n_queens import AdvancedNQueensSolver
import time

def test_basic_algorithms(n=6):
    """测试基础算法"""
    print(f"\n测试{n}皇后问题 - 基础算法")
    print("=" * 40)
    
    solver = NQueensSolver(n)
    
    # 测试DFS
    start = time.time()
    solution = solver.dfs_one_solution()
    dfs_time = time.time() - start
    print(f"DFS找到一个解: {solution} (耗时: {dfs_time:.6f}秒)")
    
    # 测试BestFS
    start = time.time()
    solution = solver.bestfs_one_solution()
    bestfs_time = time.time() - start
    print(f"BestFS找到一个解: {solution} (耗时: {bestfs_time:.6f}秒)")
    
    return dfs_time, bestfs_time

def test_advanced_algorithms(n=6):
    """测试高级算法"""
    print(f"\n测试{n}皇后问题 - 高级启发式算法")
    print("=" * 40)
    
    solver = AdvancedNQueensSolver(n)
    algorithms = [
        ("最小攻击启发式", solver.bestfs_attack_heuristic_one),
        ("前向检查启发式", solver.bestfs_forward_checking_one),
        ("中心优先启发式", solver.bestfs_center_preference_one),
        ("MRV启发式", solver.backtracking_with_mrv_one)
    ]
    
    results = []
    for name, algorithm in algorithms:
        start = time.time()
        solution = algorithm()
        elapsed = time.time() - start
        print(f"{name}: {solution} (耗时: {elapsed:.6f}秒)")
        results.append((name, elapsed))
    
    return results

def test_all_solutions(n=4):
    """测试求所有解的性能"""
    print(f"\n测试{n}皇后问题 - 求所有解")
    print("=" * 40)
    
    solver = NQueensSolver(n)
    
    # DFS求所有解
    start = time.time()
    solutions = solver.dfs_all_solutions()
    dfs_time = time.time() - start
    print(f"DFS找到 {len(solutions)} 个解 (耗时: {dfs_time:.6f}秒)")
    
    # BestFS求所有解
    start = time.time()
    solutions = solver.bestfs_all_solutions()
    bestfs_time = time.time() - start
    print(f"BestFS找到 {len(solutions)} 个解 (耗时: {bestfs_time:.6f}秒)")
    
    return dfs_time, bestfs_time

if __name__ == "__main__":
    print("N皇后算法快速测试")
    print("=" * 50)
    
    # 测试不同规模
    for n in [4, 6, 8]:
        test_basic_algorithms(n)
        test_advanced_algorithms(n)
        if n <= 6:  # 小规模的才测试所有解
            test_all_solutions(n)
    
    print("\n✅ 所有测试完成！")
```

## 🔍 测试结果分析

### 预期性能指标（N=8时）

| 算法 | 求一个解时间(秒) | 求所有解时间(秒) |
|------|-----------------|-----------------|
| DFS | ~0.0002 | ~0.0038 |
| BestFS | ~0.0003 | ~0.0045 |
| 中心优先启发式 | ~0.00003 | ~0.0040 |
| 前向检查启发式 | ~0.0005 | ~0.0077 |

### 性能分析要点

1. **求一个解的性能排名**：
   - 中心优先启发式应该是最快的
   - DFS应该性能稳定
   - 前向检查应该相对较慢

2. **求所有解的性能排名**：
   - DFS应该是最快的
   - 启发式算法由于额外计算开销，可能不如基础算法

3. **规模影响**：
   - N从4到8，时间应该增长100-300倍
   - N从8到10，时间应该增长25-50倍

## ⚠️ 常见问题解决

### 问题1：Python无法运行
**解决方案**：
- 检查Python是否正确安装
- 尝试使用完整路径：`C:\Python39\python.exe demo.py`

### 问题2：算法运行时间过长
**解决方案**：
- 减小N值（从4或6开始）
- 使用更快的算法（如中心优先启发式）

### 问题3：找不到模块
**解决方案**：
- 确保所有.py文件在同一目录
- 检查文件名是否正确
